# mock-proxy-server
 mock proxy server

```shell
# install dependencies:
  $ cd mock-proxy-server && yarn

# run the app:
  $ yarn start

# dev the app:
  $ yarn run dev

# debug the app:
  $ yarn run debug
```

.
├─bin ------ 启动脚本
├─config --- 配置项
├─data ----- 数据
├─proxy ---- 代理
├─public --- 页面资源
├─routes --- 路由
├─utils ---- 工具
└─views ---- 页面
